Stack Overflow Clone

Recreating Stack Overflow website using Rails.

* Ruby on Rails version - 4.2.1
* User Stories
  * Release 1
    * As a user, I should be able to post a question with a title and content.
    * As a user, I should be able to see a list of questions.
  * Release 2
    * As a user, I should be able to answer a question.
    * As a user, I should be able to view all the answers for a question.
    * As a user, I should be able to edit the questions I've posted.
